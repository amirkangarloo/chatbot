# ChatBot

This is a open source chatbot project

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development mode
$ pnpm run dev

# production mode
$ pnpm run prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
